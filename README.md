# VoiceTM


Private monorepo for the VoiceTM / YapMe app.


## Projects


- `VTM/` — Expo React Native frontend
- `vtm-backend/` — Node/Express backend
- `yapme-realtime/` — Cloudflare realtime worker


## Notes


This repo is private. Do not commit `.env` files or secret credentials.


## Setup


Each project has its own environment variables and run commands.
